# Крестики-Нолики

Игра крестики-нолики против компьютера, написанная на HTML/CSS/JavaScript.

## Как запустить

Открой файл `index.html` в браузере — установка не нужна.

## Возможности

- Игра против ИИ (алгоритм минимакс с альфа-бета отсечением)
- Два уровня сложности: **Легко** и **Сложно**
- Счёт между раундами
- Подсветка победной линии
- Тёмный дизайн

## Технологии

- HTML / CSS / JavaScript — один файл, без зависимостей
